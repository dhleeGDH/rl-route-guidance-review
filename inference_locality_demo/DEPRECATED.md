# DEPRECATED — not used in the paper

The inference-locality principle (Section III) is a definitional necessary condition on the
state (the dynamic, time-varying state variables must be given in predictive form for
navigation-level routing); it is established by argument and by the corpus count, not by a
performance experiment. The performance comparison this folder implemented was found, on a
fair baseline, to depend on the congestion model and did not measure the principle, so it
was removed from the manuscript. Only `experiments/boundary_open_demo/` is part of the paper.
